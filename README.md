# О проекте

Это backend API для приема лидов (форм обратной связи) и последующей пересылки или сохранения.

## Особенности

- паттерн outbox обеспечивает гарантированную пересылку форм при временной недоступности транспорта;
- Clean Arcitecture и Event Bus позволяют легко подключать новые транспортные адаптеры;
- защита от ботов с помощью гибридной reCAPTCHA v3/v2, CORS, CSRF, helmet и throttling.
